# 我的小太阳

一个可以记录生活、健康、计划、生日与力量训练的个人日历网页。

## 在线版本

网站发布后，可直接在浏览器中使用，无需安装或注册。

## 数据与隐私

- 生活记录、健康数据、生日和自定义训练保存在访问者自己的浏览器 `localStorage` 中。
- 数据不会上传到 GitHub，也不会被其他访问者看到。
- 不同浏览器和设备之间暂不自动同步。
- 清理浏览器网站数据可能会删除本地记录。

## 功能

- 公历、农历、法定节假日和调休展示
- 生活文字、图片、音乐和视频记录
- 血糖、餐食、运动和饮水记录
- 血糖趋势与生活事件复盘
- 训练方案、主训练选择和自定义训练
- 城市天气与三日预报

## 本地运行

直接打开 `index.html` 即可使用。

## 发布

项目通过 GitHub Pages 自动部署。推送到 `main` 分支后，部署工作流会更新在线网站。
